For running the Python code you would need:
1. Below libraries
	twitter-python
	pytz
	scikit learn nltk
2. twitter account
	consumer key
	consumer secret
	access token key
	access token secret

For opening the .twbx file you would need
1. Tableau Desktop software (you can download trial version from tableau's website)
2. SentimentScores.csv (included)
3. Tech-IBScore.txt (included)
4. twitter.txt, metadata.txt, words.txt (included and also can be generated by python code)
